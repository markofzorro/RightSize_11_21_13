Means

A mean or average is a way to measure the central tendency of continuous variables. You sum up the values for each individual and divide by the number of individuals.

If his temperature of 37 degrees Centegrate then x1 = 37 degrees, the second person is i = 2 and his temperature is x2, and so on.

The means "add up the x value for each of the i individuals." The number below the means start with this individual. In this equation, the i = 1 means start with the first individual. The number above the above the means "stop when you reach this number," which, in the above equation, is the total number of individuals in the population.

So means count up all the x variables for each of the i members of the population, starting with the first and stopping when you have reached the total number of individuals.
